General Information

Software requirements

  • Sublime Text 3 used to edit the .HTML, .JS, .CSS, .SCSS, .php files
  • Koala used as precompiler for SCSS to CSS convert.

Client Manager comes with a flexible file structure that can be easily used.

The following scheme displays the template files structure.

  • Documentation - contains documentation of template cusotmisation and support
  • License - contains License file of template
  • Theme - contains the main files of the theme. which will be uploaded on server.
    • App - Contains Controller, Model and Views Files
    • Builder - contains sytem files like router, session, Mailer and more.
    • Clients - contains Client Portal files(Structure is same as main Directory).
    • Config - contains configuration files.
    • Install - contains installation files(Please delete this folder after installation).
    • Public - contains CSS, JS, Images files.
    • Public - contains CSS, JS, Images files.
  • Screenshot - Contains screenshot of theme.
  • Sources - Contains screenshot of theme.
    • sass - contains .css SCSS files used in the theme.
    • DB File - contains .css Database .mysql file used in the theme.
  • readme.txt - contains the information about theme.

Installation

First time you have to configure and then you are good to go. We have created automatic installer to install Client Manager theme. Automatic installer is similar to wordpress and opencart.

1 .First method for installing theme Using Installer (Recommended)

  • Create Database and Database User in your Database and remeber credential of it.
  • Unzip the downloaded package in an empty directory and upload everything from upload folder to your server directory.
  • Open browser (like - chrome) and run your server directory URL (like - yourdomain.com) in browser. It will take you to automatic installer
  • Installer will ask database credentials and other info like mail, mobile number etc.
  • Once you enter these information. Installation will be done.

Installtion is similar to wordpress installtion. If you need more help on installation. Please send us mail on pepdevofficial@gmail.com

Note - This is automatic process. You do not have to open any code file.

2 .Second method for installing theme or Manual Process

If you are not comfortable with installer than you can go for manual process. Do not worry installer just fill database information in config.php file and load dummy data in database.

  • Create Database and Database User in your Database and remeber credential of it.
  • After that import client_manager.sql file(which is available in Source/DB file folder) in your Database which you have created.
  • Unzip the downloaded package in an empty directory and upload everything from upload folder to your server directory.
  • After that you need to put that credentials in two file(config.php).
    1. First file : This file located in config folder which is in main code folder.
    2. Second File file This file located in config folder which is in main code clients folder. See the below structure of config file.
    Please update Database information, Directory path and Web URL in these config files. See the below structure of config file.
  • Run your URL and it will open clientmanager theme.

GDPR Rule Implemented

We have implemented these rules of gdpr in our theme

  • Admin enters the client/lead details into Office Manager and assign the lead to staff/sales member. This sales-member + admin are now the only people who can access the client data, no other sales-members can view. This applies to any forms where client details are held such as projects, leads, invoices, quotes etc etc.
  • Within Leads a datefield for data destruction/expire date need to be populated by sales-member.
  • On contacts & leads there is marketing checkbox, 'None | Email | Phone | SMS | Social Media'.
  • Admin/SuperAdmin receives lead from client or 3rd party and can assign to staff.
  • Created Client Portal so client can access their data any time.
  • Client can Rectify their data.
  • Client can Edit their data.
  • Client can download their quotes and invoices.
  • Client can choose their data for marketing or restrict it.
  • If there is data breach. you can send message manually to client directly from contact view page.
  • Conset for Cookies

If there are any rules left let us know we will try to implement that rule asap.
